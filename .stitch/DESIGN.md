# Design System: Anime Tracker (Teal Cyan Professional)
**Project ID:** ANIME-TRACKER-PRO

## 1. Visual Theme & Atmosphere
The application embodies a **"Cyber-Noir"** aesthetic: deep, space-like backgrounds contrasted with vibrant, glowing functional elements. The density is **Medium-High**, prioritizing information density for "Power Users" (Otakus) while maintaining extreme legibility through glassmorphism and subtle elevation.

## 2. Color Palette & Roles
*   **Deep Space Background (#020617):** The foundational layer. Provides a noiseless, high-contrast base for all components.
*   **Vibrant Teal Action (#14B8A6):** The primary brand color. Used for progress, "Save" actions, and primary focus states.
*   **Electric Sky Accent (#38BDF8):** Used for interactive secondary elements, "Share" triggers, and animated glows.
*   **Soft Slate Secondary (#94A3B8):** Used for non-emphasized text and muted borders.
*   **Off-White Foreground (#E2E8F0):** The primary text color, chosen for low eye strain in dark environments.

## 3. Typography Rules
*   **Inter (Sans):** The primary typeface.
*   **Headers:** ExtraBold weights with tight letter-spacing (-0.02em) for a modern, "digital" feel.
*   **Body:** Medium/Regular weights for readability.
*   **Numbers/Scores:** Monospace elements (where appropriate) or semi-bold variants for clarity.

## 4. Component Stylings
*   **Buttons:** Generously rounded (12px-16px). High-vibrancy solid backgrounds for primary actions, with subtle glows on hover.
*   **Cards/Containers:** "Subtly rounded" (12px-20px) corners. Backgrounds use glassmorphic translucency (80-90% opacity) with a 1px border (#1E293B) for definition. Shadows are "Whisper-soft" diffused blacks.
*   **Inputs/Forms:** Deep dark backgrounds (#020617) with subtle borders that transition to Teal on focus.

## 5. Layout Principles
*   **Whitespace:** Generous gutter spacing (24px) between major sections.
*   **Rhythm:** Consistent vertical rhythm using multiples of 4px.
*   **Elevation:** Uses Z-index layering combined with backdrop blurs to establish clear hierarchy between the dashboard and modal overlays.
